Project Title

CI/CD Pipeline using Jenkins and Docker

Project Description

This project demonstrates a complete CI/CD pipeline using Jenkins and Docker. The application is built, tested, and deployed automatically using Jenkins, and containerized using Docker for consistent environments.

Tools & Technologies

  • Jenkins
  • Docker
  • Git & GitHub
  • Maven
  • Java Application

Pipeline Workflow

  • Code pushed to GitHub repository
  • Jenkins pulls the latest code
  • Build using Maven
  • Run tests
  • Create Docker image
  • Deploy container

Conclusion

This project helped in understanding CI/CD automation and containerization. It improves deployment speed and ensures consistency across environments.
